# meal-queen

### 사내 식권 관련 프로젝트

관리는 간편하고, 식사는 만족스럽게 임직원을 위한 맞춤형 식사를 도와주는 프로그램을 개발.

개발 환경:
* Java 17~19, Spring Boot
* intelliJ IDEA CE

## 구현하고자 하는 기능

1. 유저 기능 (로그인, 회원가입 데이터 관리 등)
2. 포인트 기능 (결제, 포인트 차감, 포인트 지급 등)
3. 관리 기능 (회사, 사원 포인트 관리)
4. 어드민 기능 (회사로 포인트 지급 및 차감 등)
**추가 중 ...**
